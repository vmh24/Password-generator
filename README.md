# Password generator
Generate a random password with the specified length.

## Usage
#### Python
**Windows:**
```bash
python pwGen.py
```
**Unix:**
```bash
python3 pwGen.py
```
#### Bash
**Without parameter:**
```bash
./pwGen.sh
```
**With parameter:**
```bash
./pwGen.sh <param1>
```
* `<param1>`: Lenght of the password
